그래픽 품질에는 영향을 미치지 않습니다.
아마 게임내에서 상점이 더 빠르게 열릴겁니다.

일반적인 LOL(모든 서버), 가레나LOL(모든 버전) 그리고 LOLQQ (중국LOL)에서 작동합니다.

라이엇 (리그오브레전드 제작자) 은 이렇게 말했습니다..


LoLUpdater는 당신을 밴당하게 하지않습니다.

롤의 DLL 파일을 Nvidia/Intel/Adobe/Microsoft 에서 가장 최신의 사용한것으로 대체하여 작동합니다. (롤의 파일은 최신버전이 아닙니다).

  • LoLUpdater.exe 을 실행하고 적절한 디렉토리를 고릅니다::

    • Regular LoL: Riot Games\League of Legends (lol.launcher.exe 가 있는곳)

    • GarenaLoL: GarenaLoL\GameData\Apps\LoL (lol.exe 가 있는곳)

    • LoLQQ (Chinese LoL): 腾讯游戏\英雄联盟 (lol.launcher_tencent.exe 가 있는곳)

수동 설치: (클릭)
    How to Unblock a file:
    Right click file -> Properties -> Unblock


  • On Windows XP you will need this to unblock files, extract and run "Unblock.bat" in the same directory as the file to be unblocked (Edit Unblock.bat and change the filename inside)

  • Install Adobe Air and NvidiaCG (Does'nt matter if you have an ATI graphics card), unblock the installers.

  • Move cg.dll, cgGL.dll and cgD3D9.dll from %WindowsDrive%:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\%Latest%\deploy

  • Copy Adobe AIR.dll from %WindowsDrive%:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0 (put NPSWF32.dll in the Resources folder)

  • Download CPU-Z and check what SIMD instructions your CPU supports then download the right one (AVX2 > AVX > SSE2): AVX2-Win8,AVX2-Win7, AVX-Win8,AVX-Win7, SSE2-Win8, SSE2-Win7, Vista or WinXP and put it in the same directory as the Cg files (see above), rename it to tbb.dll and unblock it.

  • Download msvcp120 & msvcr120 and copy them to the same place as cg.dll (See above), unblock both files.

  • You can also copy the two files in the previous step and paste them in %leaguedirectory%\RADS\projects\lol_patcher\releases\%Latest%\deploy

  • Now you are free to play! (Repeat every patch)
