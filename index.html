2015-01-27 17:49 GMT+1 (5.1 patch)
Installation instructions:
  • Exit the PvP.net client/game

  • Install the Microsoft Visual C++ Redistributable(s) (both if you are on a 64bit cpu, x86 only if you are on a 32bit cpu).

  • Right click lolupdater.exe -> properties -> unblock (If you are on XP you will need streams.exe to unblock the file)

  • Run LoLUpdater and select your League of Legends or GarenaLoL\GameData\Apps\LoL directory, wait (about 1.5 minutes on a 100/10 line with a SSD), then enjoy!

  • Redownload every LoL update (it gets deleted every time LoL gets an update)
Manual Installation: (click to see)
  • Download & Install Adobe Air and NvidiaCG (Yes, you still need this even if you have an ATI gfx card), dont forget to unblock these installers first.

  • Move Cg.dll, CgGL.dll and CgD3D9.dll from C:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\highest version number\deploy

  • Copy Adobe AIR.dll and NPSWF32.dll from C:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\highest version number\deploy\Adobe AIR\Versions\1.0 (put NPSWF32.dll in the Resources folder)

  • Check the highest SIMD instruction your processor supports (with the help of the program CPU-Z) AVX2 (Haswell processors), AVX, SSE2, SSE, none or the XP version, then download the file and put it in the same directory as the Cg files, finally rename it to tbb.dll

  • Finish everything off by unblocking tbb.dll by right clicking it -> properties -> unblock, if you are unable to see the unblock button then it already is unblocked
