Increases ingame FPS and makes the PvP.net client more responsive. Open-Source and free, written in C++.

Download LoLUpdater
Download the LoLUpdater Source-Code 2015-01-12 23:13 GMT+1
Installation instructions:
  • Exit the PvP.net client/game
  • Install the Microsoft Visual C++ Redistributable(s) (both if you are on a 64bit cpu, x86 only if you are on 32bit cpu).
  • Right click lolupdater.exe -> properties -> unblock
  • If you are on XP you will need streams.exe from sysinternals to unblock the file first to avoid any nasty ingame bugs.
  • Run from any of the paths below (your installation directory for LoL)
    • Regular League installation directory C:\Riot Games\League of Legends
    C:\Program Files (x86)\riot games
    C:\Program Files\riot games
    Garena (32-Bit cpu) C:\Program Files\GarenaLoL\GameData\Apps\LoL
    C:\Program Files\GarenaLoLPH\GameData\Apps\LoLPH
    Garena (64-Bit cpu) C:\Program Files (x86)\GarenaLoL\GameData\Apps\LoL
    C:\Program Files (x86)\GarenaLoLPH\GameData\Apps\LoLPH
Manual Installation
  • Download/Install Adobe Air and NvidiaCG
  • Move Cg.dll, CgGL.dll and CgD3D9.dll from C:\Program Files (x86)\NVIDIA Corporation\Cg\bin (you can experiment with the 64bit libraries) to C:\Riot Games\League of Legends\RADS\solutions\lol_game_client_sln\releases\highest version number\deploy
  • Copy Adobe AIR.dll and NPSWF32.dll from C:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to C:\Riot Games\League of Legends\RADS\projects\lol_air_client\releases\highest version number\deploy\Adobe AIR\Versions\1.0
  • Check the highest SMID instruction your processor supports (CPU-Z can see this) AVX2 (Haswell), AVX, SSE2, SSE or none, then download the file and put it in the same directory as the cg files, finally rename the file to tbb.dll
  • To finish everything off make sure you unblock the replaced files by right clicking on the files -> properties -> unblock, if you are unable to see the unblock button then it already is unblocked
