Installation instructions:
  • Exit the PvP.net client/game
  • Unblock the vcredists by right clicking -> properties -> unblock
  • Install the Microsoft Visual C++ Redistributable(s) (both if you are on a 64bit cpu, x86 only if you are on 32bit cpu).
  • Right click lolupdater.exe -> properties -> unblock
  • If you are on XP you will need streams.exe from sysinternals to unblock the file first to avoid any nasty ingame bugs.
  • Run from any of the paths below (your installation directory for LoL)
    • Regular League installation directory C:\Riot Games\League of Legends
    C:\Program Files (x86)\riot games
    C:\Program Files\riot games
    Garena (32-Bit cpu) C:\Program Files\GarenaLoL\GameData\Apps\LoL
    C:\Program Files\GarenaLoLPH\GameData\Apps\LoLPH
    C:\Program Files\GarenaLoLTW\GameData\Apps\LoLTW
    Garena (64-Bit cpu) C:\Program Files (x86)\GarenaLoL\GameData\Apps\LoL
    C:\Program Files (x86)\GarenaLoLPH\GameData\Apps\LoLPH
    C:\Program Files (x86)\GarenaLoLTW\GameData\Apps\LoLTW
Manual Installation
  • Download & Install Adobe Air and NvidiaCG
  • Move Cg.dll, CgGL.dll and CgD3D9.dll from C:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\highest version number\deploy
  • Copy Adobe AIR.dll and NPSWF32.dll from C:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\highest version number\deploy\Adobe AIR\Versions\1.0
  • Check the highest SMID instruction your processor supports (with the help of the program "CPU-Z") AVX2 (Haswell processors), AVX, SSE2, SSE or none, then download the file and put it in the same directory as the Cg files, finally rename it to tbb.dll
  • Finish everything off by unblocking the replaced files by right clicking them -> properties -> unblock, if you are unable to see the unblock button then it already is unblocked
