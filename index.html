Installation instructions:
  • Exit the PvP.net client/Game

  • Install Microsoft Visual C++ Redistributable 2013 (x86) & Microsoft Visual C++ Redistributable 2013 (x64)

  • Right click LoLupdater.exe -> Properties -> Unblock


  • On XP you will need this to unblock the file, extract and run "Unblock.bat" in the same directory as LoLUpdater.exe (You can edit the .bat if you are doing the manual installation)

  • Run LoLUpdater and select your League of Legends or GarenaLoL\GameData\Apps\LoL directory, wait (1 minute on a 100mbit line with a SSD), then enjoy! (Close LoLUpdater once it's done)

  • You can redownload LoLUpdater every LoL-Patch to keep the benefits active

  • GarenaLoL is independent of supported patch version (will always install, no matter what)
Manual Installation: (click)
  • Download & Install Adobe Air and NvidiaCG (Does'nt matter if you have an ATI graphics card), unblock the installers.

  • Move Cg.dll, CgGL.dll and CgD3D9.dll from %WindowsDrive%:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\%Latest%\deploy

  • Copy Adobe AIR.dll from %WindowsDrive%:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0 (put NPSWF32.dll in the Resources folder)

  • Download CPU-Z and check what SIMD instructions your CPU supports then download the right one (AVX2 > AVX > SSE2): AVX2, AVX, SSE2 or Windows XP and put it in the same directory as the Cg files (see above), rename it to tbb.dll and unblock it.

  • Now you are free to play!
