--- title: Home layout: default --- LoLUpdater is currently the best (Open-Source) tool for increasing the framerate in the game League of Legends, also makes the PvP.net client more responsive/faster.

Doesn't affect graphics quality.
Might make the ingame-shop open quicker.

DavidKNA is working on the OSX version of this patch at the moment!

Works for Regular LoL (All servers), GarenaLoL (All versions) and LoL QQ (Chinese LoL)

Riot (The makers of League of Legends) has said this about LoLUpdater.


LoLUpdater can't get you banned.

Works by replacing League's DLL files with the latest ones available from Nvidia/Intel/Adobe/Microsoft, (LoL's files are not the newest).

How to Install:
  • Install Microsoft Visual C++ Redistributable 2013 (x86)

  • Exit the PvP.net client/Game

  • Run LoLUpdater.exe and select the appropriate directory:

    • Regular LoL: Riot Games\League of Legends (where lol.launcher.exe is)

    • GarenaLoL: GarenaLoL\GameData\Apps\LoL (where lol.exe is)

    • LoLQQ (Chinese LoL): 腾讯游戏\英雄联盟 (where lol.launcher_tencent.exe is)

    wait (Under 1 minute on a 100mbit line with a SSD), then enjoy! (Close LoLUpdater once it's done)

Manual Installation: (click)
    How to Unblock a file:
    Right click file -> Properties -> Unblock


  • On Windows XP you will need this to unblock files, extract and run "Unblock.bat" in the same directory as the file to be unblocked (Edit Unblock.bat and change the filename inside)

  • Install Adobe Air and NvidiaCG (Does'nt matter if you have an ATI graphics card), unblock the installers.

  • Move cg.dll, cgGL.dll and cgD3D9.dll from %WindowsDrive%:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\%Latest%\deploy

  • Copy Adobe AIR.dll from %WindowsDrive%:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0 (put NPSWF32.dll in the Resources folder)

  • Download CPU-Z and check what SIMD instructions your CPU supports then download the right one (AVX2 > AVX > SSE2): AVX2, AVX, SSE2 or WinXP and put it in the same directory as the Cg files (see above), rename it to tbb.dll and unblock it.

  • Download msvcp120 & msvcr120 and copy them to the same place as cg.dll (See above), unblock both files.

  • You can also copy the two files in the previous step and paste them in %leaguedirectory%\RADS\projects\lol_patcher\releases\%Latest%\deploy

  • Now you are free to play! (Repeat every patch)
