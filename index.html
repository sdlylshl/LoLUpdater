--- title: Home layout: default --- LoLUpdater is currently the best (Open-Source) tool for increasing the framerate in the game League of Legends, makes the PvP.net client more responsive as well.

Doesn't affect graphics quality.
Reduces the lag when opening the in-game shop.

DavidKNA is working on the OSX version of this patch at the moment!

Works for "Regular" LoL (All servers), GarenaLoL (All versions) and LoL QQ (Chinese LoL)

Riot (The makers of League of Legends) has said this about LoLUpdater.


LoLUpdater is not considered a cheat

Works by replacing League's DLL files with the latest ones available from Nvidia/Intel/Adobe/Microsoft, (LoL's files are not the newest available).

How to Install:
  • Install Microsoft Visual C++ Redistributable 2013 (x86)

  • Exit the PvP.net client and the gameGame

  • Right click LoLUpdater.exe -> Properties -> Unblock




  • Run LoLUpdater.exe and select the appropriate directory:

    • "Regular" LoL: Riot Games\League of Legends (where lol.launcher.exe is)

    • GarenaLoL: GarenaLoL\GameData\Apps\LoL (where lol.exe is)

    • LoLQQ (Chinese LoL): 腾讯游戏\英雄联盟 (where lol.launcher_tencent.exe is)

    Let LoLUpdater run and the close the window once it's done

  • Run it like this on Windows XP:



  • Repatch with LoLUpdater every LoL-Patch (Patch by Riot) to keep LoLUpdater active (Even after the micro-patches)
Manual Installation: (click)
    How to Unblock a file:
  • Right click file -> Properties -> Unblock




  • Install Adobe Air and NvidiaCG (Yes you will still need this even if you are on ATI hardware), unblock the installers.

    • Garena/LoLQQ paths:
  • %leaguedirectory%\Game and %leaguedirectory%\Air

  • Move cg.dll, cgGL.dll and cgD3D9.dll from %WindowsDrive%:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\%Latest%\deploy

  • Copy Adobe AIR.dll from %WindowsDrive%:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\%Latest%\deploy\Adobe AIR\Versions\1.0 (put NPSWF32.dll in the Resources folder)

  • Download CPU-Z and check what SIMD instructions your CPU supports then download the right one (AVX2 > AVX > SSE2): AVX2-Win8,AVX2-Win7, AVX-Win8,AVX-Win7, SSE2-Win8, SSE2-Win7, Vista or WinXP, put it in the same directory as cg.dll (see above), rename it to tbb.dll and unblock it.

  • Download msvcp120 and msvcr120 then copy them to the same place as cg.dll (See above), unblock both files.

  • Copy both files in the previous step to %leaguedirectory%\RADS\projects\lol_patcher\releases\%Latest%\deploy

  • Repeat this process after every patch (By Riot), even the micro patches
