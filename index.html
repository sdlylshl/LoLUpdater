2015-01-28 00:23 GMT+1 (5.1 patch)
Installation instructions:
  • Exit the PvP.net client/game

  • Install vcredist_x86.exe & vcredist_x64.exe

  • Right click lolupdater.exe -> properties -> unblock (If you are on XP you will need streams.exe to unblock the file)

  • Run LoLUpdater (as non-admin) and select your League of Legends or GarenaLoL\GameData\Apps\LoL directory, wait (about 1.5 minutes on a 100/10 line with a SSD), then enjoy!

  • Redownload every LoL update (it gets deleted every time LoL gets an update)
Manual Installation: (click to see)
  • Download & Install Adobe Air and NvidiaCG (Does'nt matter if you have an ATI graphics card), unblock these installers.

  • Move Cg.dll, CgGL.dll and CgD3D9.dll from C:\Program Files (x86)\NVIDIA Corporation\Cg\bin to %leaguedirectory%\RADS\solutions\lol_game_client_sln\releases\highest version number\deploy

  • Copy Adobe AIR.dll from C:\Program Files (x86)\Common Files\Adobe AIR\Versions\1.0 to %leaguedirectory%\RADS\projects\lol_air_client\releases\highest version number\deploy\Adobe AIR\Versions\1.0 (put NPSWF32.dll in the Resources folder)

  • Check the highest SIMD instruction your processor supports (with the help of the program CPU-Z) then download any of these AVX2 (Haswell processors), AVX, SSE2, SSE, none or the XP version and put it in the same directory as the Cg files (see above), rename it to tbb.dll and unblock it.
